# 2026 剖地瓜壯騎行程手冊 MVP v1.1 公開測試版

這是一個針對 **2026 剖地瓜壯騎（八天七夜，富貴角燈塔至鵝鑾鼻燈塔）** 行程開發的手機優先（Mobile-First）行程手冊網站。專為 3480 自行車隊夥伴、扶輪社友、隊員家屬與後勤支援車人員設計，提供便利、快速且美觀的行動查詢體驗。

---

## 🛠️ 技術棧 (Tech Stack)

* **核心框架**：React 19
* **建置工具**：Vite 8 (與 `@tailwindcss/vite` 深度整合)
* **樣式工具**：Tailwind CSS v4.0 (採用最新 CSS 進口架構，極致效能與響應式排版)
* **圖示庫**：Lucide React
* **部署準備**：純前端靜態網頁設計，可輕易部署至 GitHub Pages、Vercel 或 Netlify。

---

## 🌟 第一階段 MVP 實作功能

1. **首頁 (Home)**：
   * 騎乘總覽數據統計（總里程 736 km、總爬升 11,841 m）。
   * 精心設計的「山海縱貫線」橫越地形起伏 SVG 示意圖，展現路線宏偉氣勢。
   * 8 天行程卡片快速摘要。

2. **每日行程 (Itinerary)**：
   * 支援一鍵「展開全部 / 收合全部」功能。
   * 顯示每日里程、爬升、路線 Google Maps 連結。
   * 中餐、晚餐、住宿的詳情與地圖一鍵導航按鈕。
   * 預留第二階段「隊友感言」、「影片照片」上傳區塊。

3. **食宿總覽 (Food & Lodging)**：
   * **響應式排版**：電腦端以精美對照表呈現，手機端則以卡片清單顯示，方便車友在騎乘中以單手觸控查看。
   * 包含所有已訂妥的民宿、餐廳名稱、地圖與餐標資訊。

4. **成員名單 (Members)**：
   * 依任務分組：領騎/領隊、夫妻同行卡、隊友成員、後勤支援。
   * **夫妻同行卡**：PP Roger 與 Sally 獨立於專屬的橘黃漸層精美卡片中。
   * **頭像尺寸與樣式優化**：採用 `rounded-2xl` 圓角矩形，完美保留安全帽與肩膀騎行風采；頭像分別放大至 `w-20`（夫妻同行卡）與 `w-16`（一般成員）。
   * **防禦性姓名縮寫佔位**：當成員照片尚未收集完全或載入失敗（404）時，自動生成精美的姓名縮寫背景圖，不影響視覺完整度。

5. **重要提醒 (Reminders)**：
   * 精簡收錄關鍵提醒：Day 1 礁溪秘境溫泉需備泳衣、Day 5 隙頂阿宏の家提前點餐（正確沙茶鍋版本）、Day 8 返程行李車輛與單車分流安排。

6. **費用明細 (Costs)**：
   * 整理已知的住宿費用預估（總額 106,120 元）與訂金已付明細（總額 24,660 元）。
   * 標示顯眼黃色提示：「此處僅做已知資料整理，不在此做最終分攤計算」。

## 🚀 快速啟動開發環境

若要在本地端執行或編輯專案，請確保已安裝 [Node.js](https://nodejs.org/) (建議版本 v18+)。

### 1. 安裝套件
在專案根目錄執行以下指令安裝依賴項目：
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```
啟動後在瀏覽器開啟顯示的網址（預設通常為 `http://localhost:5173/` 或 `http://localhost:5174/`）即可即時預覽。

### 3. 編譯生產環境檔案
若要產生供阿亮上傳或部署的靜態 HTML/JS/CSS 資源包：
```bash
npm run build
```
編譯完成的靜態檔案將輸出於 `dist/` 目錄下。

## 📂 檔案目錄結構描述
* `public/` - 靜態資源目錄（包含 `images/members/pp-roger.jpg` 頭像等）
* `src/` - React 原始碼目錄
  * `src/components/` - 共用 UI 元件（導航、卡片詳情等）
  * `src/pages/` - 各功能分頁
  * `src/data/tripData.js` - 八天行程、成員與提醒的靜態資料庫
  * `src/index.css` - 全域樣式與 Tailwind 主題設定
  * `src/main.jsx` & `src/App.jsx` - 入口元件與狀態/路由管理
* `package.json` - 專案依賴與腳本定義
* `vite.config.js` - Vite 編譯設定與 Tailwind 插件載入
* `implementation_plan.md` - 專案第一階段 MVP 實作與修正計畫書
* `README.md` - 本說明文件
